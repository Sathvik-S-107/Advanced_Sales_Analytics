# Advanced Sales Analytics (PostgreSQL & Power BI)

## Project Overview
This project demonstrates an end-to-end sales analytics solution using PostgreSQL and Power BI.  
It focuses on analyzing revenue, customers, and products to support business decision-making.

---

## Tools Used
- PostgreSQL (SQL joins, aggregations, window functions)
- Power BI
- DAX (custom measures and KPIs)

---

## Key Analysis Performed
- Total revenue and average order value
- Customer and segment-wise revenue analysis
- Product and category performance analysis
- Interactive dashboards with filters
